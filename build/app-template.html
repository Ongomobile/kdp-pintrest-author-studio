Publishing checklist
  1. Upload the PNG and set the destination link to your Amazon page or website — the pin link is the clickable thing, not the image.
  2. Title ≤100 chars, book title + genre keyword first. Description ≤500 chars, natural keywords, no hashtag walls.
  3. Add alt text describing the book and cover.
  4. Paste the hashtags at the end of the description (5 max).
  5. Space out the variations — don't post them all the same day.
